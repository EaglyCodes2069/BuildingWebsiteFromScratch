I still lack a few fundimental elements of understanding where I could confidently stroll through a project without having to retrace my steps to accomplish what I am trying to achieve. So from that info I worry how can I know what the acceptable amount of javascript for a site should be. My other hangup is that I struggle to remember the shapes of functions without looking them up all the time. I have not practiced javascript too any great extent yet so I hope with more practice I will begin to achieve the level of understandint that I need to be great at javascript.
Manipulating the DOM is something I have used on a small scale. I can see that it is an incredible tool for accessing my html tags etc. I look forward to understanding the DOM at a larger scale as I am sure it will help me to be able to navigate my documents in a better more efficient way.
I absolutely love github. I am sure I still have so many more things to learn about it. But I feel confident with it's main use as I feel I generally understand "why" it is used. Through learning github and keeping my project updated via git, I am learning the rules of the command line which is another thing that the more I use it the more I love using it.
